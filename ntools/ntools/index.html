    Tool version automation¶

    Tool versions in this table are automatically updated from the JSON install definitions in the dev-setup folder using the UpdateVersionsInDocs MSBuild task (C#). To update the table, run:

    nb update_doc_versions

    This will extract all tool/version pairs from every NbuildAppList entry in every *.json file in dev-setup and update the documentation table accordingly. No PowerShell script is needed or maintained for this process. The Windows dev environment has good information on how to setup a Windows dev environment.
